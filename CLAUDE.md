## ПАМЯТЬ
Весь контекст, история сессий, задачи, дизайн-стандарты — в **LightRAG**:
- MCP инструмент: `query_graph("вопрос")` — запросить
- MCP инструмент: `save_to_memory("текст")` — сохранить
- Веб-интерфейс: https://chesscoin.app/webui/

**Приоритет поиска информации:**
1. **Сначала — `query_graph` (LightRAG)** — всегда, любой вопрос
2. Не нашёл в RAG → локальные файлы (README, .claude/management/, CLAUDE.md)
3. В последнюю очередь — серверные файлы (SSH)

**Правило RAG-цикла:** Если нашёл информацию в локальных файлах или через SSH — **обязательно сохрани её в RAG** через `save_to_memory()`, чтобы в следующий раз не искать снова.

В начале сессии контекст подгружается автоматически через хук.
Если нужно больше деталей — вызови `query_graph` с конкретным вопросом.
