## 🎯 Сверхзадача проекта

ChessCoin — шахматная мини-игра внутри Telegram (Mini App).
Игроки зарабатывают внутреннюю валюту ᚙ (ChessCoin) играя в шахматы:
- против ИИ (J.A.R.V.I.S, 20 уровней)
- против живых игроков (батлы на ставку)
- в турнирах (недельных, месячных, сезонных, страновых, мировых)
- в войнах между странами (командные сражения)
- **v7.0.x: P2P биржа — купля/продажа ᚙ за TON**

Монеты хранятся на балансе. Вывод через TON (биржа, airdrop).
Реферальная система: % с побед друзей зависит от военного звания.

**Стек:**
- Backend: Node.js + TypeScript + Express + Socket.io + Prisma + PostgreSQL + Redis
- Frontend: React + Vite + TypeScript + Zustand + react-chessboard
- Bot: Python + aiogram 3
- Infrastructure: Docker Compose, nginx, Timeweb VPS + S3

## 🏗 Архитектурные решения (накопленные)

### P2P Биржа
- Минимальная цена ордера: 0.00001 TON за 1 000 000 ᚙ
- Комиссия платформы: 0.5% от totalTon
- TON идёт напрямую: покупатель → продавец (минуя сервер)
- Только 0.5% → кошелёк платформы `PLATFORM_TON_WALLET`
- Монеты ᚙ замораживаются на балансе при создании ордера
- Верификация через `boc` (proof of payment от TonConnect)
- Idempotency: повторный execute с тем же txHash → 409

### Скины в игре
- Скины создателя батла применяются для обоих игроков
- При создании сессии: `boardSkinUrl`, `pieceSkinUrl` берутся из equippedItems создателя
- Хранятся в Session модели, передаются клиентам через socket

### Сессии и игры
- Максимум 3 активных сессии на игрока
- BOT и BATTLE считаются раздельно (N2)
- Турнирные матчи: `isPrivate: true`

### Экономика
- Фаза 1: эмиссия, Фаза 2: redistributive
- Все операции с балансом ONLY через `updateBalance()`
- При заморозке ордера: `updateBalance()` с отрицательным delta + тип `EXCHANGE_FREEZE`

### Реферальная система
- Бонусы зависят от военного звания (кол-ва рефералов)
- RECRUIT (0 рефералов): 0 ᚙ бонус, 0% — показываем цель следующего ранга
- PRIVATE (10+ рефералов): 3000 ᚙ, 1% от выигрыша
- Максимум EMPEROR: 40000 ᚙ, 15%

### Магазин
- 6 вкладок в 2 ряда по 3: [Аватары|Рамки|Визуал] / [Темы|Эффекты|Биржа]
- Визуал = Доски + Фигуры (два сабтаба внутри)
- TON-кнопка сверху (подключить кошелёк)
