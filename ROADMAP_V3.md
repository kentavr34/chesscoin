> Источник целевой механики — `MECHANICS.md`. Текущее состояние — `AUDIT_2026-05-14.md`.

## Принципы движения по плану

1. **Один фикс = один коммит = один пул на prod = одно подтверждение от Кенана.**
2. Каждая фаза = **один цельный режим** или **один атомарный фикс**. Никаких
   «заодно правлю соседнее».
3. **Перед фазой**: `git tag` точки отката + дамп БД prod.
4. **После каждого коммита**: проверка реальных логов на проде +
   `curl https://chesscoin.app` + ручной smoke test того что починили.
5. Если **прошлый коммит дал регрессию** — откат, разбор, тогда следующий.
6. **Запрещено** в рамках одной фазы трогать файлы соседних фаз.

## ФАЗА 0 — Аварийные хирургические фиксы (≤30 мин, риск минимальный)

> Можно сделать прямо сейчас одной командой. Без миграций кода. **Заявка
> Кенана необходима**, потому что трогаем prod.

### 0.1. ALTER TYPE TransactionType ADD VALUE × 5
**Что:** добавить `TOURNAMENT_WIN`, `BATTLE_DONATION`, `CLAN_WAR_WIN`,
`TON_DEPOSIT`, `WALLET_UNLOCK` в enum на проде.

**Почему:** прямо сейчас cron турниров падает каждый час, донаты зрителей в
батле не работают.

**Команды:**
```sql
ALTER TYPE "TransactionType" ADD VALUE 'TOURNAMENT_WIN';
ALTER TYPE "TransactionType" ADD VALUE 'BATTLE_DONATION';
ALTER TYPE "TransactionType" ADD VALUE 'CLAN_WAR_WIN';
ALTER TYPE "TransactionType" ADD VALUE 'TON_DEPOSIT';
ALTER TYPE "TransactionType" ADD VALUE 'WALLET_UNLOCK';
```
**Риск:** нулевой. Enum только расширяется, существующие данные не трогаются.

**Проверка:** в `docker logs chesscoin_backend --since 5m` исчезает строка
`invalid input value for enum "TransactionType"`. Cron `checkTournamentResults`
доходит до `transaction.create()`.

### 0.2. `checkDailyLoginTask` — `upsert` вместо `create`
**Что:** в `backend/src/services/gameTasks.ts:171` заменить `create` на `upsert`.

**Почему:** race-condition в логах, засоряет вывод.

**Риск:** минимальный, локальная замена 1 функции.

### 0.3. Записать backup-точку перед Фазой 1
```bash
git tag backup-2026-05-14-pre-phase-1
git push origin backup-2026-05-14-pre-phase-1
ssh root@185.203.118.96 "TS=$(date +%Y%m%d-%H%M%S) && \
  docker exec chesscoin_postgres pg_dump -U chesscoin chesscoin | gzip > /root/backups/db-prod-pre-phase-1-$TS.sql.gz && \
  tar -C /opt -czf /root/backups/code-prod-pre-phase-1-$TS.tar.gz chesscoin --exclude='chesscoin/node_modules' --exclude='chesscoin/.git'"
```
